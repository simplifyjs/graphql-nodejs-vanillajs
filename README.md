# Using Vanilla JS, Node JS, Express, MongoDB using GraphQL
 
## Tools
- See [GraphQL](https://graphql.org/)
- See [Node JS](https://nodejs.org/en/)
- For Database: [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)

- Node JS
- ExpressJS
- Mongoose
- MongoDB Atlas
- CORS
- GraphQL

- Fetch API

- Bootstrap 4
